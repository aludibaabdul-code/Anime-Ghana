# Anime Ghana

Simple static site scaffold for the Anime Ghana project.

Files:
- `index.html` — homepage
- `community.html` — community hub
- `news.html` — news and updates
- `manga.html` — manga recommendations
- `about.html` — about the site
- `style.css` — basic styles

To view locally: open `index.html` in your browser.
